# Lawrence Prompt

一个用于自动化 VPS 代理搭建的 AI Prompt 项目。

## 项目简介

本项目提供了一个结构化的 AI Prompt，用于指导 Claude 或其他 AI Agent 自动完成以下任务：

- VPS 安全加固（SSH 密钥登录、防火墙配置、BBR 加速）
- 代理服务搭建（Gost Shadowsocks 转发）
- 订阅服务部署（HTTP 订阅服务器）
- 本地客户端自动配置（Clash/Shadowrocket）

## 架构

```
用户设备 (本地客户端)
    ↓
VPS (轻量 HTTP 订阅服务 + Gost SS 服务端转发)
    ↓
住宅 IP (Socks5)
    ↓
目标网站
```

## 核心特性

- **自动化部署**：从 VPS 安全配置到代理服务全自动完成
- **安全可靠**：强制 SSH 密钥登录、禁用密码登录、防火墙自动配置
- **一键订阅**：自动生成兼容 Clash 和 Shadowrocket 的订阅配置
- **本地集成**：自动唤起本地客户端并注入订阅链接
- **验证闭环**：通过环境变量验证代理出局 IP 是否正确

## 使用方式

1. 将 `vps-ip-proxy-prompt.md` 的内容提供给 Claude 或其他 AI Agent
2. 按提示提供 VPS 信息和住宅 IP 信息
3. AI 将分阶段完成整个搭建流程
4. 每阶段完成后会等待你的确认再继续

## 文档说明

| 文件 | 说明 |
|------|------|
| `vps-ip-proxy-prompt.md` | 完整的 AI Prompt 文档，包含详细的执行步骤和规则 |

## 执行阶段

1. **信息收集** - 收集 VPS 和住宅 IP 信息
2. **VPS 基础配置** - SSH 加固、防火墙、BBR 优化
3. **代理服务搭建** - Gost 安装、订阅服务部署
4. **本地端测试** - 客户端配置、代理验证

## 注意事项

- 执行高风险操作（如禁用 SSH 密码登录）前会征求确认
- 所有操作都有前置检查和后置验证
- 请确保提供的 VPS 信息准确，避免配置错误导致无法连接
